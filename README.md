# angular-starter
Full initialized front-end project based on angularJS framework

# Architecture

```
assets\
  |
  |-- js\
  |   |
  |   |-- angular.min.js
  |   |-- main.js
  |   |-- directives.js
  |
  |-- templates\
  |   |
  |   |-- entity\
  |   |-- lightbox\
  |   |-- page\
```
